# Admin Portal

Separate admin portal for managing Velvet Words content.

## Setup

1. Install dependencies:
```bash
cd admin-portal
pnpm install
```

2. Run the development server:
```bash
pnpm dev
```

The admin portal will be available at `http://localhost:8081`

## Features

- **Story Management**: Create, edit, and delete stories
- **Chapter Management**: Manage chapters for each story
- **User Management**: View user information and reading activity
- **Subscription Plans**: Manage subscription plans
- **Analytics**: View overview statistics

## Authentication

Default admin password: `admin123`

## Port

The admin portal runs on port **8081** (separate from the main app which runs on port 8080).
